Selected Work

Automated River Pollution Detection

A geospatial AI system for detecting and tracing river pollution at continental scale using satellite imagery and unsupervised learning.

Water pollution is escalating globally due to industrial discharge, agricultural runoff, and inadequate waste management. Traditional monitoring methods are slow, expensive, and spatially limited — often sampling only a few points along hundreds of kilometers of river.

This project explores whether satellite imagery combined with machine learning can automate pollution detection, identify anomalies in river systems, and ultimately support upstream source tracing.

The system prioritizes scale, reproducibility, and decision-making value over model novelty.

View technical approach

Sentinel-2 imagery was selected for its global availability, 10–20m resolution, and rich spectral bands. Using Google Earth Engine, ~3,000 river locations were sampled globally and exported as 128×128 multispectral GeoTIFF patches.

Because large-scale ground truth pollution labels do not exist, the problem was framed as unsupervised anomaly detection. A hybrid autoencoder architecture was implemented, combining:

  • CNN layers for spatial pattern learning
  • Dense layers for spectral feature compression
  • A shared bottleneck enforcing information constraint

Initial results revealed a core limitation: training on mixed clean and polluted data caused the model to learn pollution as "normal." To address this, an ensemble approach was introduced, combining Isolation Forests, Elliptic Envelopes, and statistical Z-score detection.

The resulting system now flags 10–15% of samples as anomalies, producing geospatial outputs including GeoJSON layers and pixel-level heatmaps suitable for mapping and policy analysis.
